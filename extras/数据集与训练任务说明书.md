## 1 · 概述

EchoScroll 的数据层按 **四类资产 × 四步迁移** 设计：

```
        视觉                听觉              知识              V-A 桥
     ┌───────────┐      ┌───────────┐    ┌───────────┐      ┌──────────┐
     │ 中国画 ~10K│      │ V-A 25 h │    │ RAG 30 M │      │ 自标 2 K │
     │ + 跨域情感 │      │ + 民乐 60 h│   │ 中英双语 │      │ 圆环连续 │
     └──────┬────┘      └─────┬─────┘    └─────┬────┘      └────┬─────┘
            │                 │                │                 │
            └─────────────────┴────────────────┴─────────────────┘
                                      │
                                      ▼
                              统一画作 manifest + 加工产物
```

视觉 / 听觉 / 知识三类提供 **广度**，自标 V-A pilot 提供 **准度**——这是 §3.2.1 中"四步迁移"（EmoArt 预训 → ArtELingo 跨文化 → OASIS 量表校准 → 自标 2K 微调）的数据基础。

## 2 · 数据来源方法（核心：可获得性分类）

按"是不是 Claude / 脚本一键能拿"分四档：

| 档 | 描述 | 数据来源举例 | 处理方式 |
|---|---|---|---|
| **A** | 公开 + 一键直载 | HuggingFace dataset, GitHub release, Zenodo, OSF direct | `huggingface-cli download` 或 `wget -c` |
| **B** | 开放但需爬虫 | 博物馆 Open Access API（Met / Cleveland / Smithsonian / NPM Taipei） | API 翻页 + CC0 / Chinese 过滤 + IIIF 拉图 |
| **C** | 需注册或申请 | API key（Smithsonian）、HF gated（CNPM）、Google Form（ArtEmis / ArtELingo）、Zenodo 大文件 | 人工注册 / 同意 ToU |

### 2.4 V-A 自标 Pilot（最关键，自建）

| 协议项 | 设计 |
|---|---|
| 规模 | 2,000 张 |
| 抽样 | 朝代 × 画派 × 题材 分层 |
| 标注员 | 6 名母语 |
| 重复 | 每图 ≥ 3 标 |
| 工具 | Label Studio + Russell 圆环 GUI |
| 量表 | V/A 连续 + 8 类情感 + 文化词（"萧瑟" "空寂" "庙堂" 等不存在于西方情感词的中国意境） |
| 校准 | OASIS 30 张 warm-up + Wu-273 跨模态对齐 |
| 质控 | Krippendorff α ≥ 0.6，低于则加培训轮 |
| 排期 | 4 周 |
| 预算 | 70 元 / 全程（按 8000 标注次 × 0.008 元算） |
| 附产 | 可整理为 short paper（"中国画连续 V-A 标注集"）|

**MLM 加速**：先用 GPT-4o vision 对 2K 画作预标 V-A，人工只做修订；单图时间从 3 min 压到 30 s，整体周期可压缩 50%。

### 2.5 协议合规速查（决定能否开源 release）

| License | 数据集 | 商用 | 重分发 | 衍生 |
|---|---|---|---|---|
| CC0 | Met, Cleveland, Smithsonian, OASIS, cc0-music-captioned | ✅ | ✅ | ✅ |
| CC-BY-4.0 | AmazarashiEndure, CCOM-HuQin | ✅ | ✅ 需署名 | ✅ |
| CC-BY-SA | MusicBench 3.0, 中文维基 | ✅ | ✅ 同协议 | ✅ |
| CC-BY-NC | Met essays, MTG-Jamendo, EMOPIA | ❌ | ✅ | ✅ |
| **CC-BY-NC-ND** | CCMusic 全家 | ❌ | ✅ | ❌ 不可改 |
| 学术 ToU | DEAM, PMEmo, ArtEmis, ArtELingo | 看授权 | 不可重分发 | 看授权 |

**结论**：CC0 + CC-BY 部分可开源；CCMusic 系列 ND → **训练可、衍生权重仅内部使用、对外只发音频成品**。

## 6 · 训练任务详细立项

每项给出："这个任务是什么 / 输入 / 输出 / 需要的模型 / 验收"。

### 6.1 T1 · M2 V-A 头训练

| 项 | 内容 |
|---|---|
| **任务** | 在 CLIP 图像特征上训 2 层 MLP，输出 (valence, arousal) ∈ [-1, 1]² |
| **理论依据** | 报告 §3.2.3 / §4.3：V-A 是 EchoScroll 的"画 → 情感"语义桥 |
| **输入** | (1) `$DATA/annotations/emoart_va_labels.csv` 132,895 行<br>(2) `$DATA/images/emoart/` 7,512 张本地图<br>(3) CLIP ViT-L/14 冻结特征 |
| **输出** | `va_head.pt`（state_dict + 训练历史，~3 MB）<br>`eval_metrics.json`（Pearson r + MSE）<br>`predictions_val.csv` |
| **方法** | `loss = MSE(pred, label) + 0.1 × InfoNCE_by_quadrant(pred, quad)` |
| **超参** | epochs=5, batch=256, lr=3e-4, cosine schedule |
| **模型权重** | `openai/clip-vit-large-patch14`（1.7 GB / MIT） |
| **验收** | 验证集 Pearson r：V ≥ 0.55, A ≥ 0.50 |
| **下游接口** | 拷到 `M2_affective_projection/checkpoints/va_head.pt`，主仓库自动 load |
| **代码** | `echoscroll_training_tasks/T1_m2_va_training/` |

### 6.2 T2 · MusicGen baseline 烘焙

| 项 | 内容 |
|---|---|
| **任务** | 用三种系统设定生成 8 × 3 = 24 段对比音频，证明 V-A + RAG 加成 |
| **三系统** | A: text-only "Chinese ink-wash landscape music"<br>B: A + V-A descriptors<br>C: B + top-3 RAG 检索段 = EchoScroll 全开 |
| **输入** | `paintings.json` 8 画作 + 已有 V-A 预设 + RAG context（任务内附） |
| **输出** | 24 段 12 s @ 32 kHz mono wav + `manifest.json` |
| **方法** | 不微调，仅 prompt 工程；调用 `audiocraft.MusicGen` |
| **模型权重** | `facebook/musicgen-small`（1.5 GB / CC-BY-NC 4.0） |
| **验收** | 24 wav 文件完整、可播；C 主观听感优于 A |
| **下游接口** | 丢进 Demo `assets/audio/baselines/`，给 T5 喂 FAD |
| **代码** | `echoscroll_training_tasks/T2_musicgen_baseline/` |

### 6.3 T3 · MusicGen LoRA 微调（核心算力任务）

| 项 | 内容 |
|---|---|
| **任务** | 把 MusicGen 文本编码器（T5）挂 LoRA，让模型学会古筝 / 二胡音色和技法 |
| **理论依据** | 报告 §3.2.3 + §4.3 "improvement innovation"，PEFT-LoRA |
| **输入** | `instrument_captions.jsonl` 3,193 对（或经 T6 扩到 5,000+）+ 32 kHz mono ≤30 s 音频 |
| **输出** | `adapter_config.json` + `adapter_model.safetensors`（~50 MB） |
| **方法** | LoRA r=16, α=32, dropout=0.1，目标层 q/v；CE loss on EnCodec discrete codes，conditioned on caption |
| **超参** | epochs=8, batch=4, grad_accum=8 (eff=32), lr=1e-4, max_seconds=10 |
| **模型权重** | `facebook/musicgen-small` + 自带 T5 + EnCodec |
| **VRAM** | ~14 GB |
| **验收** | 主观：`eval_samples.py` 出 4 段 vanilla vs LoRA 对比，LoRA 版"中乐味"明显增强；客观：T5 FAD |
| **协议提醒** | CCMusic CC-BY-NC-ND → **adapter 仅内部使用** |
| **下游接口** | `peft.PeftModel.from_pretrained()` 加载到 MusicGen 主仓库 |
| **代码** | `echoscroll_training_tasks/T3_musicgen_lora/` |

### 6.4 T4 · BGE-M3 真嵌入 + FAISS 索引

| 项 | 内容 |
|---|---|
| **任务** | 用真 BGE-M3 替换主仓库 hash-mock embedding，让 RAG 从"对得上"升级为"分得出朝代" |
| **输入** | `chunks.jsonl` 1,129 行 + meta（dynasty / painter / motif / source） |
| **输出** | `faiss_bge_m3.index`（dim=1024, IndexFlatIP）+ `id_map.json` + `chunks_meta.parquet` |
| **方法** | BGE-M3 dense vector + L2 归一化 + FAISS 内积检索 |
| **模型权重** | `BAAI/bge-m3`（2.2 GB / MIT） |
| **验收** | `query.py --q "Northern Song mountain landscape misty distance"` 期望 top-1 为巨然 / 范宽 / 同期山水 |
| **下游接口** | 拷 `index/*` 到 `M3_art_rag/index/`，主仓库 `ArtRAGStore.from_index()` 自动读 |
| **代码** | `echoscroll_training_tasks/T4_bge_m3_rag_index/` |

### 6.5 T5 · FAD 真嵌入评估

| 项 | 内容 |
|---|---|
| **任务** | 用 VGGish 或 OpenL3 嵌入算 Fréchet Audio Distance，替换主仓库 random-projection 兜底 |
| **输入** | T2/T7 生成音频 + DEAM 参考分布（1,802 曲） |
| **输出** | `fad_real.json`（per-group FAD + delta）|
| **方法** | 嵌入 → 均值与协方差 → `‖μ_a − μ_b‖² + Tr(Σ_a + Σ_b − 2√(Σ_a Σ_b))` |
| **模型权重** | `harritaylor/torchvggish`（280 MB）或 `openl3`（18 MB） |
| **验收** | D（全开）< C（V-A+RAG）< B（V-A）< A（baseline），差值显著 |
| **下游接口** | 数值进结题报告 §6 Evaluation + PPT 主结果柱图 |
| **代码** | `echoscroll_training_tasks/T5_fad_real_eval/` |

### 6.6 T6 · LP-MusicCaps 自动 caption 扩充

| 项 | 内容 |
|---|---|
| **任务** | 用机器 caption 给 ~3K 中国乐音频打补丁，与模板法的 3,193 对互补 |
| **输入** | 古筝 99 + 二胡 1,253 + 13-dim 1,841（≈ 3,193 段，可加 CTIS 3,974 → 7K+） |
| **输出** | `caps_extended.jsonl` + `instrument_captions_v2.jsonl`（合并版） |
| **方法** | LP-MusicCaps captioner（pretrained K-pop + Western pop），10 s 片段 → 一句机器 caption |
| **模型权重** | `seungheondoh/lp-music-caps`（1.0 GB / MIT） |
